# ESP-IDF Summary

ESP-IDF (Espressif IoT Development Framework) is the official open-source development framework for Espressif SoCs, including ESP32, ESP32-S, ESP32-C, and ESP32-H series. It runs on Windows, Linux, and macOS, providing peripheral drivers, Wi-Fi/Bluetooth networking, security libraries, FreeRTOS integration, and build tooling for creating production-grade IoT firmware applications.
